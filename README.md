# RiotApi-itemMatch

### INTRODUCE
```
Give item-tree information of your lol Champion when laning phase for every enemy champion.

기존 OP.GG에서는 라인전 상대에 따른 챔피언 아이템 트리를 제공하지 않는다.

https://www.notion.so/Opponents-lol-b16f8bc8ba6446e8913c699f71e36046
```
